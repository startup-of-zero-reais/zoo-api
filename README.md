## Primeiros passos

Comandos disponíveis:

```
// Faz o build da aplicação
make build

// Executa a aplicação
make run

// Para a aplicação
make stop
```

Primeiro de tudo execute:

```
cp .env.example .env
```

Preencha os valores das variaveis de ambiente, em seguida execute:

```
make build
```

Ao finalizar o build da aplicação, execute:

```
make run
```
